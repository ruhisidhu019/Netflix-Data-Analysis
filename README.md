# Netflix Data Analysis Project

## Overview
This project analyzes Netflix movies and TV shows using Python.  
The aim of this project is to discover insights about Netflix content such as popular genres, top countries producing content, and yearly growth of movies and TV shows.

## Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Project Features
- Data Cleaning
- Handling Missing Values
- Data Visualization
- Genre Analysis
- Country-wise Analysis
- Content Growth Analysis

## Key Insights
- Movies are more common than TV Shows on Netflix.
- Drama and International genres are among the most popular categories.
- The United States contributes the highest amount of Netflix content.
- Netflix content growth increased rapidly after 2016.
  
## Visualizations Included
- Movies vs TV Shows
- Top Genres on Netflix
- Top Countries Producing Content
- Netflix Content Growth Over Years
